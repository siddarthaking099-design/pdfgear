About PDF Gears

Privacy & Security

  • 100% Client-Side Processing - No files uploaded to servers
  • No Data Collection - Your files never leave your device
  • Secure Processing - All operations in browser memory
  • No Registration - Use immediately without accounts

Technology Stack

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • PDF Processing: PDF-lib, PDF.js
  • UI/UX: Font Awesome, CSS Grid, Flexbox
  • File Handling: HTML5 File API, Drag & Drop

Browser Compatibility

  • Chrome 60+ ✅
  • Firefox 55+ ✅
  • Safari 12+ ✅
  • Edge 79+ ✅

Limitations

  • File Size: Large files (>100MB) may cause issues
  • Complex Layouts: Some formatting may not preserve
  • Encrypted PDFs: Limited password support
  • Browser Memory: Very large files may exceed limits
